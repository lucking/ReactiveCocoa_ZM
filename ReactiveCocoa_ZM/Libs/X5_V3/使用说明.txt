

一、快速入门

    1.启动数据库：运行“启动MySQL数据库.bat”（外卖功能使用，保持窗口打开，不能关闭）
    2.启动X5平台：运行“启动X5运行平台.bat”（保持窗口打开，不能关闭）

    完成以上2步即可访问系统：http://127.0.0.1:8080/x5/m
    注意：请使用chrome或者safari浏览器

    我要使用手机或平板访问系统：
    1.手机/平板浏览器输入http://X5服务的IP:8080/x5/m
    2.即可正常访问
      提示：浏览器访问系统，本地能力不可用

    我要使用App访问系统：
    1.启动x5开发工具：运行“启动X5开发工具.bat”
    2.在开发工具右边选择“Native”下的“X5”
    3.点右键“生成本地App包”启动app生成向导
    4.向导中点“下载已生成App”进入下载页面（版本发布时已经默认生成好App）
    5.用UC等工具扫描二维码下载并安装App
    6.启动App，输入服务地址http://X5服务的IP:8080/x5/m
      提示：可以参考下面定制app方式，免输入服务地址使用

    我要订制自己的app
    1.启动x5开发工具：运行“启动X5开发工具.bat”
    2.在开发工具右边“Native”目录上点右键"新建/创建本地App"
    3.根据向导输入参数并生成App
      提示：Native/X5是一个通用App，可以输入或者扫描指定URL并打开，而定制的app打开指定的URL，不用再手工输入

二、目录结构说明

    apache-tomcat
       tomat应用服务器

    jdk
       JAVA运行环境

    model
       模型资源，开发出来的资源将存放在这里

    mysql
       mysql 数据库和管理工具，bin\mysqld.exe是mysql服务，bin\heidisql.exe是mysql管理工具
	   
    node
       node js执行程序

    runtime
       X5运行时UI服务，供Studio设计器以及调试使用

    studio
       X5建模开发工具，由studio.exe启动
	   
    tools
       compile\compileUI.bat 编译model目录下的UI和mobileUI资源
       dist\dist.bat 资源合并工具
       dist\less.bat css生成工具
